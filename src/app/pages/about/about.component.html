About the project

Dit project heet ShareMyCar. Dit is een website waar jij je auto kan delen met je vrienden en familie. Je kan dan de auto reserveren voor momenten dat jij de auto wil gebruiken. Zo kan je dus nooit tegelijk plannen en weet je altijd wie de auto bij zich heeft. Dit project heeft ook een IOS app. Deze webapplicatie en de IOS app werken samen. Je kan dus beide door elkaar heen gebruiken zonder enige problemen.
Op mijn GitHub kun je alle projecten van ShareMyCar vinden.

About the data

Onder deze tekst zie je het ERD van het ShareMyCar project. Er wordt gebruik gemaakt van 4 entiteiten. Deze hebben allemaal relaties met elkaar. De uitvoering van de relatie is verduidelijkt door de tekst die bij de lijn staat.

Aan de rechterkant van deze tekst zie je een document. Je ziet hier de Car entity. Deze heeft een lijst met referenties naar de Ride entiteit. Dit is de lijst met de naam reservations. Deze Ride heeft een destination. De destination is de Location entity. Deze is embedded in Ride. Ook heeft de Ride een User reference. Een user heeft op zichzelf een lijst met referenties naar Car. En zo blijft het rondje doorgaan.
